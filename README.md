## Space Invaders
Az 1978-ban kiadott, ikonikus játék Pythonban megírt verziója a pygame játékfejlesztői csomag segítségével.

A játék lényege, hogy kis pixel-űrlényeket kell megállítani lézerágyúval, amelyek meg akarják szállni a Földet. 
Az űrlények bombákat ejtenek a védőre és egyre lejjebb ereszkednek. A játékosnak egy vízszintesen mozgó lézerágyúval kell az idegeneket lőnie, így minél több pontot szereznie.
